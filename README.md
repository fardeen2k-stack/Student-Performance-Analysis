# 📊 Student Performance Analysis

## 📌 Project Overview

This project analyzes student performance data based on **math, reading, and writing scores**.
The goal is to understand patterns, compare performance, and generate meaningful insights using data analysis techniques.

---

## 🎯 Objectives

* Analyze student performance across different subjects
* Compare performance based on gender
* Identify patterns and relationships between subjects
* Visualize data using charts and graphs

---

## 🛠️ Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📂 Dataset

* Dataset: *StudentsPerformance.csv*
* Contains:

  * Gender
  * Race/Ethnicity
  * Math Score
  * Reading Score
  * Writing Score

---

## 🔍 Analysis Performed

* Data loading and preprocessing
* Data cleaning (checking missing values)
* Statistical analysis (mean, max, min)
* Group analysis (gender-based comparison)
* Feature creation (total score)

---

## 📊 Visualizations

* Histogram → Distribution of math scores
* Bar Chart → Average score by gender
* Heatmap → Correlation between subjects

---

## 📈 Results

* Most students scored between 60–75 in math
* Reading and writing scores are higher than math
* Male students have slightly higher math scores
* Strong correlation exists between reading and writing

---

## 🧠 Insights

* Math appears to be the most challenging subject
* Students who perform well in reading also perform well in writing
* Overall student performance is average

---

## 📌 Conclusion

* More focus should be given to improving math performance
* Additional support should be provided to low-performing students
* Teaching strategies can leverage the strong relationship between reading and writing
