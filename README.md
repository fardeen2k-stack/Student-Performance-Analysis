# Student-Performance-Analysis

## 1. Overview
This analysis explores student performance based on math, reading, and writing scores.

## 2. Key Findings
- Most students scored between 60–75 in math.
- Reading and writing scores are higher than math.
- Male students have slightly higher math scores.
- Female students show more low-score outliers.

## 3. Insights
- Math appears to be a more difficult subject.
- Reading and writing skills are strongly related.
- Overall student performance is average.

## 4. Conclusion
- Additional focus should be given to improving math performance.
- Support should be provided to low-performing students.
